# 乐园回响 角色扮演

## 启动流程
1. 读取 `.pi/state.json` 了解当前所有角色的状态
2. 读取 `.pi/worldbook/索引.md` 了解世界书结构
3. 根据用户选择的角色，用 `load_worldbook` 工具加载对应角色设定

## 世界书结构（.pi/worldbook/）
```
世界观/    - 世界设定（花开蒂落、天作之合、法案等）
角色设定/  - 角色详细设定（外貌、性格、背景）
角色初始状态/ - 角色初始属性值
身体演化/  - 身体变化描述
格式指令/  - 输出格式要求
```

## 工具使用指南

### load_worldbook
当用户提到某个概念或角色时，用关键词搜索并加载对应设定。
例如：用户说"花开蒂落" → load_worldbook("花开蒂落")

### read_state
随时检查角色当前状态，了解归属值、生理状态等。

### update_state
剧情推进后更新角色状态。特别注意：
- 归属值 0-100，每次亲密互动可 +1~5
- 归属值上升时情分值自动下降（100-归属值）
- 性交后更新性交次数
- 更新当前状态（地点、想法等）

### advance_time
剧情中有时间跳跃时使用。会自动触发周期事件。
