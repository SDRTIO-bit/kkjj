# 🗂️ 项目结构速查

```
echoes-of-paradise/
├── .pi/                          # pi 运行时目录（由 setup.mjs 生成部分内容）
│   ├── settings.json             # pi 设置 — sessionDir 让对话记录存本地
│   ├── APPEND_SYSTEM.md          # 每次对话自动注入的风格规范
│   ├── state.json                # 游戏状态数据库（由 setup.mjs 生成，不提交）
│   ├── STATE_SPEC.md             # 状态字段说明（可选参考）
│   ├── sessions/                 # 对话历史 .jsonl 文件（不提交）
│   ├── prompts/                  # 预设模板（/xxx 触发）
│   ├── skills/
│   │   └── rp/SKILL.md          # RP skill 指令 — pi 的 skill 加载入口
│   ├── extensions/
│   │   ├── rp-engine.ts          # 🔧 核心引擎：4个工具 + Web 服务器
│   │   └── rp-web/               # 🌐 前端页面（浏览器用）
│   ├── worldbook/                # 📚 世界书（由 setup.mjs 生成，不提交）
│   └── SYSTEM.md                 # pi 系统提示
├── worldbook_clean/              # 📝 世界书源文件（编辑入口）
│   ├── 世界观/                   # 世界设定（花开蒂落、天作之合等）
│   ├── 角色设定/                 # 角色详细描述（外貌、性格、背景）
│   ├── 角色初始状态/             # 角色初始数值（YAML 格式）
│   ├── 身体演化/                 # 身体变化描述
│   └── 格式指令/                 # 输出格式规范
├── setup.mjs                     # 初始化脚本 — 生成 state.json + 复制世界书
├── .github/                      # GitHub 相关（Issue 模板等）
│   └── ISSUE_TEMPLATE/
├── README.md                     # 项目介绍
├── CONTRIBUTING.md               # 贡献指南
├── CHANGELOG.md                  # 更新日志
├── LICENSE                       # 许可协议
├── 1-项目结构.md                  # 本文件
├── 2-复现.md                     # 从零复现指南（含踩坑记录）
├── 3-改造过程.md                  # 改造记录（从编辑器脚本到独立前端）
└── .gitignore
```

## 快速定位

| 你想做什么 | 找 |
|-----------|-----|
| 了解项目概览 | `README.md` |
| 添加/修改角色设定 | `worldbook_clean/角色设定/` |
| 修改角色初始数值 | `worldbook_clean/角色初始状态/` |
| 添加世界观条目 | `worldbook_clean/世界观/` |
| 改引擎逻辑（工具、Web服务） | `.pi/extensions/rp-engine.ts` |
| 改前端页面样式/逻辑 | `.pi/extensions/rp-web/` |
| 改输出风格/格式规范 | `.pi/APPEND_SYSTEM.md` |
| 看运行时游戏状态 | `.pi/state.json` |
| 看对话历史 | `.pi/sessions/*.jsonl` |
| 从零搭建 | `2-复现.md` |
| 了解改造思路 | `3-改造过程.md` |

## 文件提交规则

| 状态 | 提交到 Git | 说明 |
|------|-----------|------|
| ✅ 提交 | `worldbook_clean/`、`setup.mjs` | 源文件和脚本 |
| ✅ 提交 | `.pi/settings.json`、`.pi/SYSTEM.md` | 配置文件 |
| ✅ 提交 | `.pi/extensions/`、`.pi/skills/`、`.pi/prompts/` | 扩展和 prompt |
| ✅ 提交 | `.pi/APPEND_SYSTEM.md` | 风格规范 |
| ❌ 忽略 | `.pi/state.json` | 运行时生成 |
| ❌ 忽略 | `.pi/worldbook/` | 由 setup.mjs 生成 |
| ❌ 忽略 | `.pi/sessions/` | 对话历史，含隐私 |
