# 项目结构速查

> 最后更新: 2026-05-23

## 顶层目录

```
项目根目录/
├── .pi/                              # pi 项目配置与数据
│   ├── settings.json                 # pi 设置（sessionDir 动态切换）
│   ├── APPEND_SYSTEM.md              # 常驻风格规范（每次对话自动注入）
│   ├── state.json                    # 全局游戏状态（cardStates 格式）
│   ├── state_history.jsonl           # 状态变更历史
│   ├── sessions/                     # 对话记录
│   │   └── <卡名>/                   # 按卡片隔离
│   ├── skills/rp/SKILL.md            # RP Skill 指令
│   ├── cards/                        # 📇 角色卡仓库
│   │   ├── registry.json             #   卡片注册表
│   │   ├── <卡名>/                   #   单张卡片完整目录
│   │   │   ├── worldbook/            #     世界书
│   │   │   ├── state.json            #     角色状态
│   │   │   ├── config.json           #     卡片配置
│   │   │   ├── APPEND_SYSTEM.md       #     风格规范
│   │   │   ├── regex_scripts/        #     原始正则（备份）
│   │   │   ├── tavern_scripts/       #     原始酒馆（备份）
│   │   │   ├── regex_hooks.json      #     渲染钩子
│   │   │   ├── variable_schema.json  #     变量定义
│   │   │   ├── remote_urls.json      #     远程 URL 清单
│   │   │   ├── POST_IMPORT_TODO.md   #     导入后待办
│   │   │   ├── ui/                   #     前端 UI（AI 本地化）
│   │   │   ├── scripts/              #     本地脚本（AI 本地化）
│   │   │   ├── assets/               #     静态资源（AI 本地化）
│   │   │   └── vectors/              #     向量索引（AI 本地化）
│   │   └── ...
│   ├── extensions/
│   │   ├── rp-engine/                # 🔧 引擎模块
│   │   │   ├── index.ts              #   入口（事件注册 + 用户轮数计数）
│   │   │   ├── types.ts              #   类型定义（无硬编码角色列表）
│   │   │   ├── utils.ts              #   工具函数
│   │   │   ├── registry.ts           #   可注册式 Tool/Command 注册表
│   │   │   ├── state-store.ts        #   状态持久化（多卡片 + 旧格式迁移）
│   │   │   ├── card-manager.ts       #   卡片管理器（注册/激活/查询）
│   │   │   ├── worldbook.ts          #   世界书加载（动态关键词索引）
│   │   │   ├── regex-processor.ts    #   正则脚本处理器
│   │   │   ├── author-note.ts        #   Author Note 作者注
│   │   │   ├── system-prompt.ts      #   系统提示构建（多卡片融合）
│   │   │   ├── periodic-events.ts    #   周期事件
│   │   │   ├── tools.ts              #   AI 工具（4 个）
│   │   │   ├── commands.ts           #   用户命令（5 个）
│   │   │   ├── tui-panels.ts         #   TUI 终端面板
│   │   │   ├── rp-web-server.ts      #   Web 服务器（HTTP + WS）
│   │   │   ├── tests.test.ts         #   测试
│   │   │   └── README.md             #   引擎模块详细文档
│   │   └── rp-web/                   # 🌐 前端界面（15 个文件）
│   │       ├── rp-web.html           #   主页面
│   │       ├── rp-web-app.js         #   主逻辑（WS + 卡片 UI 加载）
│   │       ├── rp-web-message-renderer.js  # 消息渲染 + 正则钩子
│   │       ├── rp-web-xml.js         #   XML 解析
│   │       ├── rp-web-state.js       #   状态管理器
│   │       ├── rp-web-settings.js    #   设置面板
│   │       ├── rp-web-markdown.js    #   Markdown 渲染
│   │       ├── rp-web-style.css      #   样式
│   │       └── ...
│   └── worldbook/                    # 世界书（运行时软链接）
├── setup.mjs                         # ⭐ 导入脚本（14 步自动流程）
├── .rpconfig.json                    # 运行时配置
├── .gitignore
├── package.json
├── README.md
└── 项目规程.md
```

## 核心文件速查

| 你想做什么 | 找 |
|-----------|-----|
| 导入角色卡 | `node setup.mjs --import <path>` |
| 管理卡片 | `/card list` / `/card activate` |
| 改引擎逻辑 | `.pi/extensions/rp-engine/` |
| 改前端页面 | `.pi/extensions/rp-web/` |
| 看/改卡片状态 | `.pi/cards/<卡名>/state.json` |
| 看变量定义 | `.pi/cards/<卡名>/variable_schema.json` |
| 看正则钩子 | `.pi/cards/<卡名>/regex_hooks.json` |
| 看远程 URL | `.pi/cards/<卡名>/remote_urls.json` |
| 看导入待办 | `.pi/cards/<卡名>/POST_IMPORT_TODO.md` |
| 看对话历史 | `.pi/sessions/<卡名>/*.jsonl` |
| 改风格/格式规范 | `.pi/cards/<卡名>/APPEND_SYSTEM.md` |
