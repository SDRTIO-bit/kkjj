# 输出格式指令

主体内容格式强调，下面的内容必须输出，但不强制按照顺序编写，每个xml标签之间可插入其他xml标签，如果标签顺序冲突则自行调整顺序确保每个标签能够输出:
<thinking>
    <进行创作思路构思与分析，禁止直接输出>
</thinking>

<content>
    <主要故事情节>
</content>

<perspective>
    <toggle_title>副视角</toggle_title>
    <content_html>
        <p>（另一视角的并行故事内容）</p>
    </content_html>
</perspective>

<UpdateVariable>
    <Analysis>
        <分析每个变量更新的理由
        <变量路径: 是否更新 (是/否) >
    </Analysis>
    <!指令格式进行变量更新
<_.set('路径', '旧值', '新值'); // 简要原因>
</UpdateVariable>
